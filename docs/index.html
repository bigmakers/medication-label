Windows (.exe) / macOS (.app) 対応

⚠️ Windowsをお使いの方へ - 初回起動時のご案内

初めてアプリを起動すると、Windows Defender SmartScreenの警告画面が表示されることがあります。
これは「発行元が未確認」のアプリに対して表示される一般的な保護機能です。
安全にお使いいただけますので、以下の手順で起動してください。

  1. 「WindowsによってPCが保護されました」という画面が表示されます
  2. 画面内の「詳細情報」というテキストリンクをクリックします
  3. すると「実行」ボタンが表示されますので、それをクリックします
  4. 2回目以降は、この画面は表示されません

※ 「詳細情報」をクリックしないと「実行」ボタンが見えない仕様になっています。ご注意ください。
